# Responsive Lecture Timetable

A sleek, modern, and fully mobile-responsive university lecture timetable layout. This project focuses on keeping the HTML structure minimal and semantic while letting advanced CSS handle the heavy lifting for layout architecture and design.

## Features

- **CSS Grid Layout:** Desktop view utilizes a robust 7-column grid structure, managing time slots and subject placements entirely via CSS coordinates.
- **Mobile Responsive:** Built with CSS Media Queries to seamlessly shift from a wide horizontal desktop grid to a clean, vertically stacked list view on mobile devices.
- **Modern UI Styling:** Features dark-themed headers, subtle zebra-row striping for better readability, and distinct color-coded blocks for different lecture modules.

## Tech Stack

- **HTML5:** Semantic, lightweight, and framework-free markup.
- **CSS3:** Advanced Grid, Flexbox, and Media Queries.

## How to View

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Resize your browser window or inspect in mobile mode to see the responsive layout adaptation.
